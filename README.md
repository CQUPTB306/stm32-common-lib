# stm32常用库

本仓库用于存放B306实验室常用的库，方便团队成员共享和协作，提高开发效率。

## 📂 目录结构
```
├── docs/            # 文档与使用说明
├── examples/        # 示例代码
├── User/            # 具体的模块代码
│   ├── Inc/         # 代码头文件
│   ├── Src/         # 代码实现文件
│   └── ...          # 其他模块
└── README.md        # 仓库说明
```

## 🚀 现有文件

```
├── STM32CommonLib/          # stm32一些常用库                  
│   ├── DoubleBufferUartDMA/     #双缓冲区串口dma发送库
│   ├── VOFA/                    #VOFA库
└── README.md                # 仓库说明
```
